Authentification
=======

Connexion
^^^^^^^^^
Afin de ce connecter, vous aurez besoin d'un compte.
Il faudra alors, entrez votre identifiant ainsi que votre mot de passe.


Création de compte
^^^^^^^^^^^^^^^^^^
Pour créer un compte, il sera nécessaire de cliquer sur "Créer un compte".

.. image:: /Images/CreationCompte.png


Il sera nécessaire de remplir toutes les données afin que votre compte soit créé.
Lors du clique sur "Inscription", votre compte sera créé et vous serez redirigé vers la page de connexion, avec comme indication le succès de votre création de compte.



Problème de connexion
^^^^^^^^^^^^^^^^^^^^^
Si vous n'arrivez plus à vous connecter a votre compte, il est possible de cliquer sur "Des problèmes pour vous connecter ?"
Dès lors, deux choix s'offrent : 

.. image:: /Images/ProblemeConnexion.png


Lors de l'oubli du mot de passe, il vous sera nécessaire de renseigner le nom d'utilisateur du compte, la question secrète ainsi que la réponse adéquate, et le nouveau mot de passe.
Si les informations renseignées sont bonnes, le mot de passe du compte sera donc changé.

Lors de l'oubli de votre nom d'utilisateur, il vous sera nécessaire de renseigner votre adresse e-mail, votre mot de passe, la question secrète ainsi que la réponse adéquate.
Si les informations renseignées sont bonnes, vous serez alors redirigé sur la page de connexion avec votre nom d'utilisateur qui sera indiqué.
